# Club América - Sitio Web Escolar

## Descripción del Proyecto
Sitio web informativo sobre el Club América desarrollado como proyecto escolar. El sitio presenta información sobre el equipo, jugadores, historia y un formulario de contacto.

## Problemática que Resuelve
Este proyecto busca centralizar información del Club América de manera accesible y atractiva para los aficionados, proporcionando:
- Información histórica del club
- Datos de jugadores actuales
- Medio de contacto con los administradores del sitio

## Objetivo del Desarrollo
Crear una plataforma web informativa que permita a los usuarios conocer más sobre el Club América, su historia, plantilla actual y establecer comunicación mediante un formulario de contacto.

## Tipo de Sitio
**Sitio Web Estático Informativo**

Es la elección más conveniente porque:
- Contenido principalmente informativo
- No requiere actualizaciones constantes en tiempo real
- Fácil de mantener y actualizar
- Bajo costo de hosting
- Rápido tiempo de carga

## Tecnologías Utilizadas

### Frontend
- **HTML5**: Estructura del contenido
- **CSS3**: Estilos y diseño visual
- Diseño responsive con meta viewport

### Estructura del Proyecto
```
club-america/
│
├── index.html          # Página principal
├── jugadores.html      # Información de jugadores
├── historia.html       # Historia del club
├── contacto.html       # Formulario de contacto
└── cs.css             # Estilos CSS
```

## Características Principales

### Páginas Implementadas
1. **Inicio (index.html)**
   - Bienvenida al sitio
   - Información general del club
   - Colores representativos
   - Filosofía del club
   - Últimos logros

2. **Jugadores (jugadores.html)**
   - Plantilla 2025
   - Porteros
   - Defensas
   - Información de cada jugador

3. **Historia (historia.html)**
   - Orígenes del club (1916)
   - Evolución de colores y escudo
   - Épocas doradas

4. **Contacto (contacto.html)**
   - Formulario de contacto
   - Campos: Nombre, Correo, Mensaje

## Principios UI/UX Aplicados

1. **Navegación Consistente**: Menú de navegación presente en todas las páginas
2. **Jerarquía Visual**: Uso de encabezados y tamaños de fuente apropiados
3. **Colores del Club**: Amarillo (#ffcc00) y Azul (#002b7f) como colores principales
4. **Responsive Design**: Adaptable a diferentes tamaños de pantalla
5. **Legibilidad**: Fuente Arial, contraste adecuado
6. **Feedback Visual**: Efectos hover en enlaces

## Instalación y Uso

### Requisitos Previos
- Navegador web moderno (Chrome, Firefox, Safari, Edge)
- Editor de código (VS Code, Sublime Text, etc.)

### Instalación Local
1. Clonar el repositorio:
```bash
git clone https://github.com/[tu-usuario]/club-america.git
```

2. Navegar al directorio:
```bash
cd club-america
```

3. Abrir index.html en tu navegador
